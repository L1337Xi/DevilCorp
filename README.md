# DevilCorp OWASP Top 10 (2025 Edition)
![DevilCorp Banner](DevilCrop.jpeg)    

**DevilCorp** is an cybersecurity training portal designed to help learners master the OWASP Top 10 (2025) through hands-on labs and real-world scenarios.
This platform simulates a multi-layered organization targeted by an elite hacker named **"Jam3s"**.

The lab features 10 unique, interactive pratical missions based on the **OWASP Top 10:2025 (Release Candidate)** standards.

## 🔍 Key Features
*   **10 Interactive Hacking Labs:** Real-world scenarios involving IDOR, 403 Bypasses, Supply Chain

## 🛠️ Lab Categories (OWASP 2025)

1.  **A01: Broken Access Control** - Exploit IDOR in admin panels.
2.  **A02: Security Misconfiguration** - Bypass proxy filters using malformed paths.                  
3.  **A03: Software Supply Chain** - Identify backdoors in 3rd party modules.                          
4.  **A04: Cryptographic Failures** - Decode weak sensitive data transmissions.                        
5.  **A05: Injection** - Perform advanced UNION-based SQLi.                                            
6.  **A06: Insecure Design** - Exfiltrate metadata to bypass reset logic.                              
7.  **A07: Authentication Failures** - Intercept 2FA codes via IDOR.                                   
8.  **A08: Software & Data Integrity** - Manipulate serialized session cookies.                        
9.  **A09: Security Logging & Alerting** - Dig through massive logs for leaks.                         
10. **A10: Mishandling Exceptions** - Trigger crashes to leak system env vars.
